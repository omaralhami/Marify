Marify Key Management

How to manage keys:

  1. Generate new keys using the button below
  2. Copy the generated key
  3. Add the key to keys.json in your GitHub repository
  4. Set the key's value to null to make it available for use
  5. Commit and push the changes to activate the key
Example keys.json format: 
{
    "used_keys": {
        "ABCDEF-GHIJKL-MNOPQR": null,
        "STUVWX-YZABCD-EFGHIJ": "existing-user-sid"
    }
}
